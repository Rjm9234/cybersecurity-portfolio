# Home Security Lab Setup

## Overview
A personal cybersecurity + AI/ML lab for hands-on learning.

## Tools

### Mac (host machine)
- Burp Suite Community — web app testing
- Python 3 — scripting and ML (requests, scapy, sklearn, pandas, numpy, matplotlib)
- Wireshark — packet analysis

### Kali Linux (ARM64 via UTM)
- Nmap 7.94 — network scanning
- Metasploit — exploitation framework
- Wireshark — packet capture

## Kali VM
- Virtualization: UTM on Apple Silicon (M4)
- Image: Kali Linux 2023 ARM64
- Default credentials: kali/kali (change after setup)

## Python Toolkit
- requests — HTTP automation
- scapy — packet crafting
- scikit-learn — ML models
- pandas — data analysis
- numpy — numerical computing
- matplotlib — visualization
